使用 Vue 创建个人博客

约 148 字小于 1 分钟

markdownvuevuepress

2025-06-18

本文将介绍如何使用 Vue.js 搭建一个个人博客。

1. 环境准备

确保你已经安装了 Node.js 和 npm/yarn。

2. 创建 VuePress 项目

npm init vuepress-app my-blog
cd my-blog
npm install

3. 配置 VuePress

编辑 .vuepress/config.js 文件，配置主题和插件。

4. 编写博客内容

在 docs 目录下创建 Markdown 文件来编写你的博客文章。

5. 部署

将你的博客部署到 GitHub Pages 或其他静态网站托管服务。
